# Sample Files

These files are used to demonstrate Cyber Shield's
primary static analysis capabilities.

⚠ None of these files are executed.
⚠ All analysis is read-only and byte-based.

## Files
- benign.txt — harmless content
- benign_script.py — safe script
- suspicious_powershell.ps1 — execution abuse
- encoded_payload.txt — obfuscation example
- lolbin_certutil.txt — LOLBin misuse
- credential_dump_hint.txt — credential access indicators
